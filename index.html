1 Harmonic analysis for what?

  • Harmonic analysis [1] is a mathematical tool that allows expressing a function f (t) in relation to a set of orthogonal functions gi (t), by means of a linear combination of these. In other words, f(t) = ∑ iaigi(t).

  • By conveniently choosing the set of orthogonal functions we can perform an analysis of f(t) according to the characteristics or properties of the functions gi(t).
  • One of the most frequent practical applications of Fourier analysis is the representation of signals according to their frequency components. This is achieved because the base functions in the harmonic analysis are sinusoids.

2 The Fourier trigonometric series

  • Any function (or signal) s(t) can be represented by a a linear combination of sinosuidal functions.
  • Let f(t) be a function deﬁned in the interval (t0,t0 + 2π ω0 ). The Fourier trigonometric series allows to represent f(t) in terms of the complete orthogonal set of sine-wave functions [2]
    {1; cos(ω0t),cos(2ω0t),⋯,cos(nω0t),⋯; sin(ω0t),sin(2ω0t),⋯,sin(nω0t),⋯}

    through the linear combination (stf)

    f(t) = a0 + ∑ n=1∞a n cos(nω0t) + bn sin(nω0t), (stf)

    where

    t0 < t < t0 + 2π ω0,

    ω0 is the fundamental frequency component expressed in radians per second and a0, {an} and {bn} are the coeﬃcients of the Fourier trigonometric series.

  • Note that this summation can only reproduce the behavior of a function (or a signal) in an interval (of time) equal to
    2π ω0.

    As just deﬁned, this interval is equal to the period of the lowest frequency component that exists in the signal.

2.1 Fourier coeﬃcients of the trigonometric series

  • The coeﬃcients of the Fourier trigonometric series quantify the correlation between the function f(t) and the corresponding basis functions (the amount of each of the “pure sinusoidal signals” that must be added together to obtain the analyzed signal).
  • Mathematically, they are calculated as the proportion that exists between the energy of the correlation of the signal with the respective sinusoidal function (an) and the energy of that sinusoidal function (bn), ie
    an = ∫ t0t0+Tf(t)cos(nω 0t)dt ∫ t0t0+T cos2(nω 0t)dt (an)
    bn = ∫ t0t0+Tf(t)sin(nω 0t)dt ∫ t0t0+T sin2(nω 0t)dt (bn)

    where

    T = 2π ω0.
  • For n = 0, coeﬃcient
    a0 = 1 T∫ t0t0+Tf(t)dt,

    is the average value of f(t) in the interval (t0,t0 + T). a0 is refered as the direct current component or DC (Direct Current) of f(t) in such interval.

  • On the other hand, knowing that
    ∫ t0t0+T cos2(nω 0t)dt = ∫ t0t0+T sin2(nω 0t)dt = T 2 ,

    the Eqs. an and bn can be also written as

    an = 2 T∫ t0t0+Tf(t)cos(nω 0t)dt (anOK)

    and (bnOK)

    bn = 2 T∫ t0t0+Tf(t)sin(nω 0t)dt, (bnOK)

    which is usually the most common in the literature [1, 2].

3 The Fourier exponential series

  • Fourier exponential series is a more compact representation of Eq. stf, written as a function of the complex exponential.
  • Be the deﬁnitions
    a0 = F0 an = Fn + F−n bn = j(Fn − F−n), (defs_Fes)

    where j = −1.

    If we substitute these deﬁnitions in the Equation stf, we get that

    f(t) = F0 + ∑ n=1∞(F n + F−n)cos(nω0t) + (j(Fn − F−n))sin(nω0t).

    Multiplying we get to that

    f(t) = F0 + ∑ n=1∞F n cos(nω0t) + F−n cos(nω0t)+ jFn sin(nω0t) − jF−n sin(nω0t).

    Operating

    f(t) = F0 + ∑ n=1∞F n cos(nω0t) + jsin(nω0t)+ F−n cos(nω0t) − jsin(nω0t).

    Now applying the trigonometric changes

    ejnω0t = cos(nω0t) + jsin(nω0t) e−jnω0t = cos(nω0t) − jsin(nω0t) (e_sin_cos)

    in the previous expression we get that

    f(t) = F0 + ∑ n=1∞F nejnω0t + F −ne−jnω0t.

    By operating with the sign of the variable n and partially undoing the sum we get to that

    f(t) = F0 + ∑ n=1∞F nejnω0t + F −nej(−n)ω0t = F0 + ∑ n=1∞F nejnω0t + ∑ n=−1−∞F nejnω0t.

    Finally, joining the two summations and the term that was left out we have to

    f(t) = ∑ n=−∞∞F nejnω0t. (Fes)
  • The Fes equation is known as the Fourier exponential series and the Fn as the coeﬃcients of that series.

3.1 The Fourier complex coeﬃcients

  • Using the deﬁnitions made in the Expression defs_Fes it is possible to easily ﬁnd that
    Fn = 1 2(an − jbn). (F_n)
  • Substituting this eq. in the expressions anOK and bnOK we get that (F_n_OK)
    Fn = 1 2 2 T∫ t0t0+Tf(t)cos(nω 0t)dt − j 2 T∫ t0t0+Tf(t)sin(nω 0t)dt = 1 T∫ t0t0+Tf(t)cos(nω 0t)dt − j∫ t0t0+Tf(t)sin(nω 0t)dt = 1 T∫ t0t0+Tf(t)e−jnω0tdt (F_n_OK)

4 Representation of a periodic function using the Fourier serie in the interval (−∞,∞)

  • So far we have found that it is possible to represent any function f(t) in a ﬁnite interval (t0,t0 + T) by any of its Fourier series. Outside that interval, f(t) and its Fourier series are not necessarily equal.
  • However, if the function is periodic then its serial representation can be applied to the entire interval (−∞,∞). This can be easily demonstrated if we take into account that the base functions are periodic since
    ejπω0t = ejπω0(t+T),

    where, T = ω0 2π. This result implies that the Expressions stf and Fes (both series) are valid for the entire interval (−∞,∞) when f(t) is periodic.

5 The Fourier complex spectrum

  • The development in Fourier series of a periodic function is really equivalent to the transformation of the function in terms of its angular frequency components
    ω0,2ω0,3ω0,⋯

    being ω0 = 2π T and T the period of said function. Therefore, we have two equivalent representations of the same function: that of the time domain and that of the frequency domain.

  • A representation in the frequency domain, that is, when we indicate the amplitudes of the diﬀerent frequency components is what we call spectrum, which for all periodic functions is discrete (functions are reconstructed from their frequency components using a summation, not an integral).
  • When we use the exponential Fourier series (Fes equation) the angular frequency components are
    0,±ω0,±2ω0,⋯

    and the corresponding amplitudes of the complex spectrum1

    F0,F±1,F±2,⋯.

    Since these magnitudes are complex, they can also be described in terms of their magnitude (or modulus) and phase.

6 Representation of a function in the interval (−∞,∞): the Fourier transform

  • At this point we have learned to represent any function f(t) in terms of an exponential (or trigonometric) series in a ﬁnite interval, and that for the special case in which this function is periodic, its representation can be extended to the entire interval (−∞,∞).
  • Knowing this, to deal with non-periodic signals we can use the fact that any non-periodic function can be considered as such (that is, periodic) if we assume that it has an inﬁnite period, that is, if T →∞.
  • Let fT(t) be a function of period T and let f(t) be a non-periodic function. We deﬁne fT(t) so that
    limT→∞fT(t) = f(t),

    that is, it ceases to be periodic only when its period tends to inﬁnite. Because fT(t) is periodic, its exponential Fourier series is of the form (fT) (see Expression Fes)

    fT(t) = ∑ n=−∞∞F nejnω0t, (fT)

    where the Fourier coeﬃcients (using the Expression F_n_OK) are

    Fn = 1 T∫ t0t0+Tf T(t)e−jnω0tdt. (Fn2)
  • The term Fn represents the amplitude of the frequency component nω0, where we recall, the fundamental frequency ω0 = 2π T . When T increases, ω0 decreases and the spectrum becomes denser. It also occurs (see Equation Fn2) that the amplitude of each component is reduced.
  • When T = ∞, the Fn become inﬁnitely small but there is also an inﬁnite number of spectral components. Under these conditions, the spectrum exists for any value of ω and is no longer a discrete spectrum but continuous. We will highlight this idea with the change of notation
    nω0 = ωn,

    with which the equations fT and Fn2 remain as

    fT(t) = ∑ n=−∞∞F nejωnt

    and

    Fn = 1 T∫ t0t0+Tf T(t)e−jωntdt.

    Since Fn is a function of ωn, we will also change the notation

    Fn = Fn(ωn)

    with what these equations are written as

    fT(t) = ∑ n=−∞∞F n(ωn)ejωnt

    and

    Fn(ωn) = 1 T∫ t0t0+Tf T(t)e−jωntdt.

    Finally, be it by deﬁnition

    Fn(ωn) = F(ωn) T ,

    with which Equations fT and Fn2 ﬁnally remain as

    fT(t) = 1 T∑ n=−∞∞F(ω n)ejωnt (fT2)

    and (Fn3)

    F(ωn) = ∫ −∞∞f T(t)e−jωntdt. (Fn3)

    (Recall that the integration range is now (−∞,∞)).

  • If we substitute T = 2π ω0 in fT2 we obtain that
    fT(t) = 1 2π∑ n=−∞∞F(ω n)ejωntω 0. (fT3)

    As we have already indicated, when T →∞, fT(t) = f(t), ω0 → 0 and there is an inﬁnite number of terms ωn. Therefore, the discrete sum of the Equation fT3 becomes an integral (with which the subscripts on n disappear) and ω0 must be represented by dω. Thus, Equation fT3 remains as

    f(t) = 1 2π∫ −∞∞F(ω)ejωtdω (iFt)

    and the Eq. Fn3 as

    F(ω) = ∫ −∞∞f(t)e−jωtdt. (Ft)
  • The function F(ω) represents the amplitudes of the inﬁnite spectral components of f(t) and is known as the spectral density function.
  • The equations iFt and Ft are known as the pair of Fourier transforms. It is said that Ft is the direct Fourier transform of f(t) and that iFt is the inverse Fourier transform of F(ω).
  • Note that, according to the Fourier transform, when the signals are aperiodic their spectrum is continuous (dense). On the contrary, the Fourier transform of a periodic signal (which coincides with its representation by one of the Fourier series) generates a discrete spectrum (see Equation Fes).
  • Summing up: Fourier reasoned that a signal without period can be considered as periodic if it is supposed an inﬁnite period [2]. More precisely, in the Fourier series representation of a periodic signal, as the period increases, the fundamental frequency decreases and the harmonically related components2 in the frequency become closer. As the period becomes inﬁnite, the frequency components form a continuum and the Fourier series sum becomes the integral [1].

7 Representation of a signal in the time and frequency domains

  • If the function (or signal) F(ω) is the (direct) Fourier transform of the function (or signal) f(t), what we will notice by
    ℱ[f(t)] = F(ω),

    then F(ω) represents the relative amplitudes of the diﬀerent complex exponential components. Both representations (that of the time f(t) and that of the frequency F(ω)) uniquely specify the same function (or signal).

  • As we already know (Eq. Ft), in general F(ω) is complex, that is, its representation module-argument is
    F(ω) = |F(ω)|ejϕ(ω),

    where |F(ω)| is the modulus or magnitude of F(ω) and is calculated as

    |F(ω)| = Re (ω)2 + Im (ω)2,

    and where ϕ(ω) is the argument or phase ofF(ω) and is calculated as

    ϕ(ω) = arctan Im(ω) Re(ω),

    being

    F(ω) = Re(ω) + jIm(ω).

    Therefore, in general, both diagrams (the module and the phase) are needed to graphically represent F(ω).

8 The spectrum module of real functions is symetrical

  • In the signal analysis, all the Fourier transforms that are made are necessary to real functions (without imaginary component).
  • When f(t) is real, then
    F(ω) = F∗(−ω),

    and therefore

    |F(ω)| = |F(−ω)|,

    that is, the spectrum module of f(t) is symmetric with respect to the frequency 0.

Proof

  • If we apply the Change e_sin_cos in the Equation Ft we will obtain that
    F(ω) = ∫ −∞∞f(t)cos(ωt)dt − j∫ −∞∞f(t)sin(ωt)dt.

  • On the other hand, using the same change but for − ω
    F(−ω) = ∫ −∞∞f(t)cos(ωt)dt + j∫ −∞∞f(t)sin(ωt)dt.

  • Therefore, it holds that
    F(ω) = F∗(−ω).

10 The Dirac (unit impulse) function

  • The unit impulse function [1] plays a determining role in the theory of signal communication and in particular in the sampling theorem. It is deﬁned as:
    and it fulﬁlls that
    ∫ −∞∞δ(t)dt = 1 si t = 0 0 otherwise, (delta_1)

    that is, although it is an inﬁnitely narrow pulse, it has an area of 1 because it has inﬁnite amplitude. For the same reason we have, by deﬁnition, that (delta_2)

    ∫ −∞∞δ(t)f(t)dt = f(0)∫ −∞∞δ(t)dt = f(0), (delta_2)

    and in the same way, that

    ∫ −∞∞δ(t − t 0)f(t)dt = f(t0). (delta_3)
  • By virtue of these deﬁnitions we could say that the unit impulse function is able to calculate the value of a function at the point where it is deﬁned (the delta).

13 Fourier transform of the complex exponential function

  • The Fourier transform of the complex exponential function of frequency ω0 is a unit impulse of energy 2π in ω0,
    ℱ[ejω0t] = 2πδ(ω − ω 0). (FTCE)

Proof 1

18 Fourier transform of a time-shifted signal

  • If
    ℱ[f(t)] = F(ω)

    then

    ℱ[f(t − t0)] = F(ω)e−jωt0 . (ℱ[f(t − t0)])

    That is, moving a function over time is equivalent to multiplying in the domain of the frequency by the complex exponential function.

Proof

By deﬁnition of Fourier transform we have that

ℱ[f(t − t0)] = ∫ −∞∞f(t − t 0)e−jωtdt.

Be x = t − t0. Then

ℱ[f(t − t0)] = ∫ −∞∞f(x)e−jω(x+t0)dx = ∫ −∞∞f(x)e−jωxdx ⋅ e−jωt0 = F(ω)e−jωt0.

19 Inverse Fourier transform of a time-shifted function

  • If
    ℱ[f(t)] = F(ω)

    then

    F(ω − ω0) = ℱ[f(t)ejω0t]. (F(ω − ω0))

    That is, moving the spectrum of a function is equivalent to multiplying that function in the time domain by a complex exponential.

Proof

By deﬁnition of Fourier transform one has to

ℱ[f(t)ejω0t] = ∫ −∞∞[f(t)ejω0t]e−jωtdt = ∫ −∞∞f(t)e−j(ω−ω0)tdt (Por deﬁnición de transformada de Fourier para ω = ω − ω0) = F(ω − ω0).

21 The time-domain convolution theorem

  • It establishes that the convolution of two functions f1(t) and f2(t) in the time domain is equivalent to multiplying their spectra F1(ω) and F2(ω), that is,
    f1(t) ∗ f2(t) = ℱ−1[F 1(ω)F2(ω)]. (ConvT)

Proof

By deﬁnition of the Fourier transform and the convolution operation one has to

ℱ[f1(t) ∗ f2(t)] = ∫ −∞∞∫ −∞∞f 1(τ)f2(t − τ)dτe−jωtdt = ∫ −∞∞f 1(τ)∫ −∞∞f 2(t − τ)e−jωtdt︸ F2(ω)e−jωτdτ.

Note that

∫ −∞∞f 2(t − τ)e−jωtdt = ℱ{f 2(t − τ)}

and appliying the Expression ℱ[f(t − t0)] we get that

∫ −∞∞f 2(t − τ)e−jωtdt = F 2(ω)e−jωτ.

Therefore,

ℱ[f1(t) ∗ f2(t)] = ∫ −∞∞f 1(τ)F2(ω)e−jωτdτ = F2(ω)∫ −∞∞f 1(τ)e−jωτdτ = F1(ω)F2(ω).

22 The frequency-domain convolution theorem

  • It establishes that the multiplication of two functions f1(t) and f2(t) in the time domain is equivalent (except for a scale factor) when convolving its F1(ω) and F2(ω) spectra, i.e.
    f1(t)f2(t) = ℱ−1 1 2πF1(ω) ∗ F2(ω). (ConvF)

Proof

By deﬁnition of the inverse Fourier transform (Eq. iFt)

ℱ−1 1 2πF1(ω) ∗ F2(ω) = 1 2π∫ −∞∞ 1 2πF1(ω) ∗ F2(ω)ejωtdω

By deﬁnition of convolution (Eq. f1(t) ∗ f2(t))

= 1 2π∫ −∞∞ 1 2π∫ −∞∞F 1(τ)F2(ω − τ)dτejωtdω

rearranging

= 1 2π∫ −∞∞F 1(τ) 1 2π∫ −∞∞F 2(ω − τ)ejωtdωdτ.

If we use now the Eq. F(ω − ω0) and we apply the inverse Fourier transform we get to that

1 2π∫ −∞∞F 2(ω − τ)ejωtdω = f 2(t)ejτt.

Therefore, substituting this expression in the previous equation we have to

ℱ−1 1 2πF1(ω) ∗ F2(ω) = 1 2π∫ −∞∞F 1(τ)f2(t)ejτtdτ

rearranging

= f2(t) 1 2π∫ −∞∞F 1(τ)ejτtdτ

applying, again, the inverser Fourier transform (Eq. iFt)

= f2(t)f1(t).

23 Convolution of a function with the unit impulse function

The convolution of a function f(t) with the unit impulse function δ(t) results in the same function f(t). That is to say,

f(t) ∗ δ(t) = f(t).

Proof

As we know, by the convolution theorem in time

f(t) ∗ δ(t) = ℱ−1[F(ω)Δ(ω)].

We also know about Eq.FTδ that Δ(ω) = 1, so necessarily

f(t) ∗ δ(t) = ℱ−1[F(ω)] = f(t).

24 Convolution with the time-shifted unit impulse function

  • The convolution of a function f(t) with the unit impulse function displaced in time δ(t − t0) results in the same function f(t) displaced in time. That is to say,
    f(t) ∗ δ(t − t0) = f(t − t0). (f(t) ∗ δ(t − t0))

Proof

Using the theorem of convolution in the time domain (Eq. ConvT) and the Eq. ℱ[f(t − t0)] we get that

f(t) ∗ δ(t − t0) = ℱ−1F(ω)(Δ(ω)e−jωt0) = ℱ−1(F(ω)e−jωt0)Δ(ω) (teniendo en cuenta, de nuevo, la Eq. ℱ[f(t − t0)]) = ℱ−1ℱ[f(t − t0)]Δ(ω)︸1 = ℱ−1ℱ[f(t − t0)] = f(t − t0).
